# Spoonfeeder

NestJS project scaffolder with 113 composable recipes. Generate production-ready projects with RFC-compliant error handling, composable database/auth/cloud/deploy recipes, and AI assistant context.

## Quick Start

```bash
git clone <repo-url>
cd spoonfeeder
pnpm install
pnpm build
node dist/index.js
```

## Features

- 7 project types: HTTP API, AWS Lambda, Microservice, CLI App, Scheduled Worker, Monorepo, Full-Stack
- 113 composable recipes (database, auth, security, cloud, observability, deployment, CI/CD)
- RFC 9457 compliant error handling
- AI context generation (Claude Code, Cursor, GitHub Copilot)
- Nx generators for post-scaffolding recipe management (add/remove/migrate)
- Every generated project ships with test scaffolds
