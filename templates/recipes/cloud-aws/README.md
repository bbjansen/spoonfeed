# AWS Cloud Recipes

Amazon Web Services integrations for NestJS applications.

## Recipes

| Recipe                                | Description                                                  |
| ------------------------------------- | ------------------------------------------------------------ |
| [CloudFront](./cloudfront/)           | CDN integration with signed URLs                             |
| [CloudWatch](./cloudwatch/)           | Ship application logs to CloudWatch Logs                     |
| [Cognito](./cognito/)                 | User pool authentication and authorization                   |
| [DynamoDB](./dynamodb/)               | NoSQL database integration                                   |
| [ElastiCache](./elasticache/)         | Managed Redis caching service                                |
| [EventBridge](./eventbridge/)         | Event-driven architecture with EventBridge                   |
| [RDS](./rds/)                         | Managed database connection with IAM authentication          |
| [S3](./s3/)                           | Object storage for file uploads and downloads                |
| [Secrets Manager](./secrets-manager/) | Retrieve and cache secrets                                   |
| [SNS](./sns/)                         | Pub/sub messaging with Simple Notification Service           |
| [SQS](./sqs/)                         | Reliable message queuing with Simple Queue Service           |
| [SSM Parameter Store](./ssm/)         | Configuration retrieval from Systems Manager Parameter Store |
