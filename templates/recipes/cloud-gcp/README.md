# GCP Cloud Recipes

Google Cloud Platform integrations for NestJS applications.

## Recipes

| Recipe                                | Description                                          |
| ------------------------------------- | ---------------------------------------------------- |
| [Cloud CDN](./cloud-cdn/)             | Content delivery with signed URL support             |
| [Cloud Functions](./cloud-functions/) | Serverless NestJS workloads                          |
| [Cloud Logging](./cloud-logging/)     | Stackdriver logging integration                      |
| [Cloud SQL](./cloud-sql/)             | Managed database connection                          |
| [Cloud Storage](./cloud-storage/)     | Object storage for files                             |
| [Firebase Auth](./firebase-auth/)     | User identity management with Firebase               |
| [Firestore](./firestore/)             | NoSQL document database                              |
| [Memorystore](./memorystore/)         | Managed Redis caching service                        |
| [Pub/Sub](./pubsub/)                  | Messaging integration for event-driven architectures |
| [Secret Manager](./secret-manager/)   | Secret storage and retrieval                         |
