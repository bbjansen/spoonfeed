| [Cosmos DB](./cosmos-db/)               | NoSQL database                     |
| [Entra ID](./entra-id/)                 | Authentication (formerly Azure AD) |
| [Front Door](./front-door/)             | CDN and global load balancer       |

| [Key Vault](./key-vault/)               | Secret and key management          |
| [Service Bus](./service-bus/)           | Enterprise messaging               |
| [SQL Database](./sql-database/)         | Managed SQL database connection    |
